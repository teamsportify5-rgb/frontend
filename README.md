# Sportify Management System - Frontend

React + Vite frontend with shadcn/ui components.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create a `.env` file in the frontend directory:
```env
VITE_API_URL=http://localhost:8000
```

3. Run the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173`

## Build

To build for production:
```bash
npm run build
```

## Features

- **Authentication**: Login and JWT-based authentication
- **Dashboard**: Overview with summary cards and recent activity
- **Orders Management**: Create, update, delete, and track orders
- **Attendance**: Check-in/check-out and view attendance records
- **Payroll**: Generate payroll and download PDF salary slips

## Tech Stack

- React 18
- TypeScript
- Vite
- TailwindCSS
- shadcn/ui components
- React Router
- Axios
